# 測試資料夾說明

這個資料夾用於存放要評估的 Excel 檔案。

## 使用方式

1. 將您的 Excel 測試檔案（.xlsx 或 .xls）放入此資料夾
2. 在 Streamlit 介面的側邊欄選擇「📂 本地資料夾」
3. 從下拉選單中選擇您要評估的檔案
4. 點擊「🚀 執行評估」按鈕

## 檔案要求

- 檔案格式：Excel (.xlsx 或 .xls)
- 檔案名稱：避免使用特殊字符
- 臨時檔案：以 ~ 開頭的檔案會被忽略
